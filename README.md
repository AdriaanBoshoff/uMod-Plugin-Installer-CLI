# uMod-Plugin-Installer-CLI
Getting tired of manually searching and downloading plugins? Just place this in your plugins folder and run it. It allows you to search and install plugins from a command line.

Linux Requirements:

    libcurl4
    openssl

Tested on:

    Windows 10
    Debain 9
    Ubuntu 18.04
